# PDF Editor

A JavaScript-based PDF editing application that enables users to modify PDF documents by editing text content and managing signatures.

## Features

- Add, remove, and update text in PDF documents
- Add and remove signatures (image, drawn, typed)
- Preserve document formatting
- Load and save PDF files

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Testing

```bash
npm test
```

## Technology Stack

- **pdf-lib**: Core PDF manipulation
- **PDF.js**: PDF rendering
- **fast-check**: Property-based testing
- **Jest**: Testing framework
- **Webpack**: Bundling
